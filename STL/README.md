#### Depressor Arms configuration options:
There are three depressor arm configuration options included in this repo: 
1. Standard Voron gantry (used with both with and without gantry backers).
2. Monolith gantry
3. Siboor Trident 9MM gantry

#### Depressor Arms BOM:
**Standard Voron Gantry (no backers) and Monolith Gantry versions**
- 2x M3x10 BHCS to fix the depressor arm base to the gantry
- 2x M3x14-M3x18 BHCS to fix the depressor arm to the depressor base
- 2x M3x5x5 heatsets (standard voron spec). Fit from the rear of the depressor arm base.
- 2x M3 washers for the depressor base

**Standard Voron Gantry (with backers)**
- 2x M3x12 BHCS to fix the depressor arm base to the gantry
- 2x M3x14-M3x18 BHCS to fix the depressor arm to the depressor base
- 2x M3x5x5 heatsets (standard voron spec). Fit from the rear of the depressor arm base.
- 2x M3 washers for the depressor base
